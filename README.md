# Analisis-de-Componentes-Principales-PCA-y-Clasificacion-de-Tumores-de-Mama
Este proyecto de Machine Learning utiliza el dataset de cáncer de mama de Scikit-learn para demostrar la efectividad de la Reducción de Dimensionalidad mediante el Análisis de Componentes Principales (PCA), y cómo esta técnica optimiza la visualización y la clasificación de tumores en malignos y benignos.
